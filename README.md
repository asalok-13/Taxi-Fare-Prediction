# Taxi-Fare-Prediction
This project builds a machine learning regression model to predict taxi fares based on trip-related features such as pickup/drop-off locations, distance, time, and passenger count.
It demonstrates the complete data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, model training, evaluation, and explainability using Python.
